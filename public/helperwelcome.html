SightCall Helper

Welcome to Helper

Congratulations! You have successfully installed Sightcall Helper - a help-desk mini-application.

Agent Interface

Agents log on to Helper and await customer-service requests. Go to the root of the project an log on to become an agent. The default Agent names and passwords are as shown below.

AGENT LOGIN

The defaults are:

  • agent1 / a1password
  • agent2 / a2password
  • agent3 / a3password
  • agent4 / a4password

Visitor Interface

Website Visitors requiring assistance click on a "Video Assistance" button that you embed in your website. This will launch the "Visitor" page, with an optional query parameter suggesting a skill that a qualified Agent should provide. An example is shown below.

/visitor/wait?skill=loans
